RubyKaigi 2 日目。  
少し前日のお酒が残る頭で何とか起床して会場に向かいました。

![1](/assets/images/posts/20220909/1.jpg)

Contribute to Ruby [@matz](https://github.com/matz)  
https://www.youtube.com/watch?v=m_LW5WIYJ9Q&ab_channel=RubyKaigi

Ruby の開発に対して何か私も出来ることからしていきたいという気持ちを新たにしました。  
このところはインプットに重点を置いてきていたので、そろそろ溜め込んできたものを外に出していければいいなと思いました。

Ruby に関わるツールの改善の分野は非常に興味があるので、そこで何か自分も取り組めたら良さそうに思いました。

Do Pure Ruby Dream of Encrypted Binary Protocol? [@unasuke](https://github.com/unasuke)  
https://www.youtube.com/watch?v=hCos6p_S-qc&ab_channel=RubyKaigi

Ruby で QUIC という通信プロトコルを実装するために進めてきたことを紹介されていた。  
とても難しそうだと話を聞いていて思ったのですが、お一人でこれを進められいるのは凄いです。

![2](/assets/images/posts/20220909/2.jpg)

Make RuboCop super fast [@koic](https://github.com/koic)  
https://www.youtube.com/watch?v=bMLy09UV6TY&ab_channel=RubyKaigi

RuboCop Server Mode のお話でした。  
導入された時の Pull Request が上がっていることを見てはいたのですが、ちゃんとパッチの内容を見れていなかったので知れてよかったです。

これで、普段のコーディングの時はグッと実行が速くなるので良さそう。

その後はお昼ごはん。今日はローストビーフ弁当をいただきました。  
お肉が柔らかくて美味しい...感動でした。

![3](/assets/images/posts/20220909/3.jpg)

Afternoon Break では伊勢茶をいただいて、フルーツをいただきました。  
いちじく久しぶりに食べたのですが、とても美味しくて感動しました。

![5](/assets/images/posts/20220909/5.jpg)

そして、ブースに行きブースに立っていました。  
ブースに Matz が来てくださったので一緒に写真を撮らせていただきました。

感謝をしっかりと伝えることができて本当に良かったです。

![4](/assets/images/posts/20220909/4.jpg)

ブースエリアの人が落ち着いたタイミングで、午後からは 2 つのセッションを拝聴しました。

ポケモンカードと題材は身近なものなのですが、実装について考えないといけないことが多く難しそうだ...と思いながら聞いていました。
