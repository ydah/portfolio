title: Fix doc formatting written in unescaped file path
date: 2022-03-07
layout: post
---

RuboCop のドキュメントを見ていてどうやら `foo/**/*` のようなパス表記のところが壊れているらしいことに気づいて修正を行った。  
ドキュメントは[AsciiDoc](https://asciidoctor.org/docs/what-is-asciidoc/)というもので書かれているらしい。

するとどうやら、AsciiDoc では `*/*` のようにアスタリスクで挟むと強調表示になるようで、エスケープが必要らしいことが分かったので対応した。

Fix doc formatting written in unescaped file path  
https://github.com/rubocop/rubocop/pull/10444

最初は直接 `*.adoc` ファイルを修正して PR を出したのだが、どうやらそれではダメでジェネレータの修正が必要とのことだった。  
https://github.com/rubocop/rubocop/blob/master/lib/rubocop/cops_documentation_generator.rb

ジェネレータ側の修正を行ってやり取りをしたところマージされた。  
RuboCop の各 extension もこのジェネレータを使ってドキュメントを生成しているので、次のバージョンから修正されると思う。
